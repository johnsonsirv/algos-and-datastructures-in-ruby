## Installation

> Clone the repository to your local machine

```sh
$ git clone https://github.com/johnsonsirv/repl.it.git
```

## Usage

> Go to the directory

```sh
$ cd repl.it
```

> Run any of the files

```sh
$ ruby binary_search.rb
```
